How to embed the chat widget on any external website

üìã Integration Steps

Step 1: Replace the Server URL

Change localhost:5000 to your actual chat server URL:

<!-- Replace localhost:5000 with your server URL --> <script src="http://YOUR_CHAT_SERVER_URL/embed.js"></script>

Step 2: Examples of Valid URLs

<!-- Local development --> <script src="http://localhost:5000/embed.js"></script> <!-- Production domain --> <script src="https://chat.yourcompany.com/embed.js"></script> <!-- IP address --> <script src="http://192.168.1.100:5000/embed.js"></script> <!-- Subdomain --> <script src="https://support.example.com/embed.js"></script>

Step 3: Add to Your Website

Place the script tag anywhere in your HTML, preferably before the closing </body> tag:

<!DOCTYPE html> <html> <head> <title>Your Website</title> </head> <body> <!-- Your website content --> <h1>Welcome to our website</h1> <!-- Chat widget - replace with your server URL --> <script src="http://YOUR_CHAT_SERVER_URL/embed.js"></script> </body> </html>
